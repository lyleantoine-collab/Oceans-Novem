# Erosion & Runoff Control Table

| Slope % | Rainfall intensity | Recommended method               | Runoff reduction |
|---------|--------------------|----------------------------------|------------------|
| 0–5 %   | Any                | Direct broadcast + mulch         | >98 %            |
| 5–15 %  | <50 mm/h           | Hydroseed slurry + tackifier     | 95–97 %          |
| >15 %   | Any                | Biodegradable geotextile + granules | 99 %+          |

Tested in Newfoundland winter 2024–2025.
