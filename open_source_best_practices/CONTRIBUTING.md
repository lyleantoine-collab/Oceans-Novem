# Contributing to Ocean’s Novem

1. Fork → branch (e.g., nhung-nghiem/strain-tweak) → PR to main.
2. Tests: Run qPCR sims in /tests/ before PR.
3. Indigenous lens: Tag @lyleantoine-collab for FPIC review.
4. No GMOs without ethics addendum.

Issues? Use labels: type:enhancement, needs:fpic, good-first-issue.
